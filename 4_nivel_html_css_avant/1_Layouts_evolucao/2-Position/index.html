para ver o efeito de posicionamento "sticky", aplica o "position: sticky" e arraste a barra de rolagem deste contêiner.

O elemento irá rolar junto com contêiner até que esteja no topo dele (ou alcance o deslocamento especificado em "top") e então erá para de rolar.
